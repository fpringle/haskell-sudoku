Basic solving techniques

genInitialOptions :: Sudoku -> SudokuWithOptions #

Convert from Sudoku to SudokuWithOptions, eliminating obviously impossible options.

eliminateOptions :: SudokuWithOptions -> SudokuWithOptions #

eliminate options using a basic method: if a cell at position pos has only one option x, delete all occurrences of x from cells in the same row, column or box as pos

eliminateOptionsRepeatedly :: SudokuWithOptions -> SudokuWithOptions #

Repeatedly apply eliminateOptions until there are no changes.

scanRows :: SudokuWithOptions -> SudokuWithOptions #

Scan each row to see if there is only 1 place where a value x could be - if there is, place it there

scanCols :: SudokuWithOptions -> SudokuWithOptions #

Scan each column to see if there is only 1 place where a value x could be - if there is, place it there

scanBoxes :: SudokuWithOptions -> SudokuWithOptions #

Scan each box to see if there is only 1 place where a value x could be - if there is, place it there

scan :: SudokuWithOptions -> SudokuWithOptions #

Apply scanRows, scanCols and scanBoxes

scanRepeatedly :: SudokuWithOptions -> SudokuWithOptions #

Repeatedly apply scan until convergence.
