type Sudoku = Grid Int Source #

Sudoku is a 2D 9x9 list representing the contents of a partially- or totally-filled sudoku grid. A blank space is represented by a 0.

type Pos = (Int, Int) Source #

Pos is an (Int, Int) tuple representing the position of a cell in a sudoku grid.
